## 📦 Local Development

### Prerequisites

- Node.js
- npm

### Setup

```bash
# Clone the repository
git clone https://github.com/mmori7/portfolio

# Navigate to project directory
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```
